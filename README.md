# regex-generator# regex-generator | 正则表达式生成器

自然语言描述需求 → AI生成正则表达式 → 浏览器本地实时测试验证 → 一键导出多语言代码片段。

## 功能特点

- **生成 + 验证闭环**：AI生成正则后，粘贴真实数据立即看到匹配高亮结果，不用肉眼debug
- **本地测试零成本**：匹配验证完全在浏览器端用JS正则引擎执行，不调用AI，不消耗token，改一个字符即时刷新结果
- **多语言代码导出**：一键生成 Python / JavaScript / Java / PCRE 对应的可直接粘贴调用代码
- **常用模板库**：内置手机号、邮箱、身份证、URL、IP地址等8个高频正则模板，点击即可加载示例数据并体验测试功能，无需API Key
- **AI微调反馈**：生成结果不满意时，可标注具体哪几行测试数据不对，AI据此二次调整（最多3轮）
- **ReDoS熔断保护**：测试执行放在独立Web Worker中运行，主线程设1秒超时，超时自动终止，避免灾难性回溯正则冻死页面
- **深浅色主题切换**，本地记忆偏好

## 技术说明

- 接口统一走 `https://jotarou.com/v1/chat/completions` 网关转发，不直连任何大模型官方API
- 测试验证基于浏览器 JS 正则引擎，与 Python / Java / PCRE 等目标语言的实际运行环境可能存在语法细节差异（如命名捕获组写法），重要场景请自行复核
- 无外部CDN依赖，纯前端单文件实现

## 使用方式

打开页面后：
1. （可选）右上角🔑图标填入在 jotarou.com 注册获取的 API Key，用于AI生成功能
2. 描述你的正则需求，或点击场景标签快速填入示例描述
3. 选择目标语言后点击"生成正则"
4. 在右侧粘贴测试数据，实时查看匹配结果
5. 满意后点击"导出代码片段"，复制到你的项目中使用

不填API Key也可以直接点击底部"常用模板"体验完整的测试验证与代码导出功能。
